# ShopEase – E-Commerce Web App

This is a full stack e-commerce project that I built using the MERN stack.  
In this project, users can register, login, browse products, add items to cart, and place orders.  
There is also an admin panel where admin can manage products and orders.

---

## Live Link
https://shopease-psi-eight.vercel.app

---

## Features

- User authentication (Login / Register)
- Product listing
- Add to cart
- Order placement
- Order history
- Admin dashboard
- Role-based access (user / admin)

---

## Tech Stack

- Frontend: React (Vite), Tailwind CSS  
- Backend: Node.js, Express.js  
- Database: MongoDB  
- Authentication: JWT  
- Deployment: Vercel (Frontend), Render (Backend)  

---

## How to run locally

Clone the project:

git clone https://github.com/Sauravkumar8684/SHOPEASE.git

Go to backend folder:

cd backend  
npm install  
npm start  

Go to frontend folder:

cd frontend  
npm install  
npm run dev
